La seconde grande catégorie de développeur est le développeur web. Ce dernier produit des lignes de code pour créer des sites internet, des sites intranet ou des applications mobiles, qui fonctionnent sur un navigateur web. Il existe trois grands types de développeurs web : le développeur front-end, le développeur back-end et le développeur full-stack
